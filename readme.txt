Проект Decbase

1. Встановлення.

    git clone - команда для копіювання репозиторію.

2. Запуск.

    У вас повинен бути встановлений Live Server. 
    Для встановлення його в редакторі VS Code, зайдіть у вкладку Extensions і скачайте пакет "Live Server".
    Запустіть проект нажавши на кнопку нижньої панелі "Go live".

3. Особливості.

    При запуску активується Preloader-анімація на 5 секунд. Далі відкривається сайт.
    
    Якщо користувач неактивний протягом 1 хвилини то отримує повідомлення про допомогу.
    Якщо користувач хвилину не реагує на повідомлення то отримує повідомлення про вихід.

    На сайті є 12 секцій.

        1. Header.
            Має навігаційну панель. Навігаційні кнопки переносять користувача до відповідної секції на сторінці.
            При нажатті на логотип, повернемось на початок сторінки.
        2. Modern Interior.
            Інформаційна секція.
        3. What We Do.
            Секція з пропозиціями.
            Є 4 кнопки
                All відображає по одному пункту з трьох напрямів.
                Інші кнопки відображають пропозиції свого напрямку.
            Дані підтягуються за допомогою функції fetch при запуску сторінки, модифікуються та відображаються.
        4. About Us.
            Інформаційна секція.
        5. 20 Years Of Successful Working The Market.
            Інформаційна секція.
        6. Experience.
            Інформаційна секція з результатами діяльності компанії.
        7. Testimonials.
            Секція з відгуками клієнтів.
            Реалізований безкінечний Slider за допомогою Bootstrap 5.
            Має три сторінки, по 2 відгука на кожну.
            * у зв'язку з накладанням Bootstrap стилів на стилі проекту було використано !important до відповідних css властивостей.
        8. Latest news.
            Інформаційна секція.
        9. Sign_up.
                Реалізована форма на введення ім'я, прізвища та пошти.
                Реалізована валідація введених даних, з візуальною інформацією про невалідність даних.
                При введені імені "Sigma" і валідності інших даних отримаєте приємний бонус.
            Умови валідації:
                Для імені та прізвища всі букви мають бути латинського алфавіту, перша буква - велика. 
                Для пошти відповідна валідація email.
        10. Координати.
                Реалізована робота з API.
                На сторінці 2 google карти. Перша показує вид зверху, інша -  функція показу вулиць тих самих координат.
        11. Footer.
                Реалізовані посилання для майбутнього розширення сайту.
                Також є працюючі посилання на соцмережі.
        12. Copyright.
                Вказана дата в секції підтягується з годинника браузера.
